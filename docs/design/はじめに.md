# はじめに

版数: 第1.0版

本資料は、現場試験アプリの第1.0版設計資料である。Sync Agentを中心に、Labonity ProviderおよびLiberty Providerの設定解決、既存DB読取、クラウド同期、OCR確認結果登録を定義する。

## 本資料の位置づけ

本資料は、現場試験アプリ第1.0版設計資料一式の導入である。共通基本原則は`00_資料体系・レビューガイド_第1.0版.md`および`99_統合設計仕様書_第1.0版.md`を正とし、本資料では参照順と主要範囲を示す。

## 読み方

最初に`00_資料体系・レビューガイド_第1.0版.md`、次に`01_要件定義書_第1.0版.md`、`02_基本設計書_第1.0版.md`、`07_詳細設計書_第5分冊_同期機能・データベース定義_第1.0版.md`を確認すると、Provider設定とDB登録先の全体像を把握できる。
